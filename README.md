# Lark Bots

Feishu/Lark bot projects built with [lark-cli](https://github.com/anthropics/lark-cli) and Claude.

## Projects

### [feishu-claude-mvp](./feishu-claude-mvp/)

A lightweight bridge that connects Feishu IM to a local Claude CLI session — chat with Claude directly in Feishu with streaming replies, LaTeX formula rendering, multi-turn sessions, and GLM/third-party backend support.

→ [Full README](./feishu-claude-mvp/README.md)

## Tech Stack

- **Runtime**: Node.js (ESM), TypeScript
- **IM Transport**: lark-cli (Feishu CLI)
- **AI Backend**: Claude CLI (supports Anthropic-compatible APIs)
